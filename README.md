# Database for recruitment agency
Описание: компания занимается подбором персонала топ-уровня. Необходимо разработать структуру БД, а также продемонстрировать работу с тестовыми данными. Задача: разработать структуру, оформить ER-диаграмму с указанием связей между таблицами, написать скрипты для создания таблиц, заполнить таблицы сгенерированными тестовыми данными, описать типовые SQL-запросы.

Для удобства чтения подготовила файл pdf с пояснениями и описанием БД, таблиц, со скринами ERD. Также подготовила sql-файлы со скриптами создания, наполнения БД и запросами (JOIN, сложенные запросы, представления, хранимые процедуры и тригеры)
